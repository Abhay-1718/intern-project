# User Management Application

A simple user management application built with React, Vite, and Tailwind CSS. This application allows you to create, read, update, and delete user data using the JSONPlaceholder API.

## Features

- Fetch and display a list of users in a responsive table.
- Add new users with form validations.
- Edit existing user details.
- Delete users with confirmation.
- Responsive design that works well on both desktop and mobile.
- Stylish user interface using Tailwind CSS.

## Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **Vite**: A fast frontend build tool.
- **Tailwind CSS**: A utility-first CSS framework for styling.
- **Axios**: A promise-based HTTP client for making API requests.
- **react-router-dom**: For routing in the application.
